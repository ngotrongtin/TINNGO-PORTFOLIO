Tóm tắt nghề nghiệp

Sinh viên ngành Công nghệ thông tin với kinh nghiệm lập trình sử dụng Ruby on Rails và C++. Đã thực hiện nhiều dự án cá nhân bao gồm phát triển ứng dụng web tích hợp Stripe API và OmniAuth, cùng với các dự án lập trình nhúng và game. Đam mê công nghệ và mong muốn tìm kiếm cơ hội thực tập hoặc làm việc trong môi trường phát triển phần mềm.

Kỹ năng

  • Ngôn ngữ lập trình: Ruby, C++, Javascript cơ bản
  • Framework: Ruby on Rails
  • Công nghệ web: Stripe API, OmniAuth
  • Lập trình nhúng, Iot: C++
  • Lập trình game: SFML
  • Cơ sở dữ liệu: PostgreSQL
  • Kỹ năng mềm: Giao tiếp, tiếng anh cơ bản, giải quyết vấn đề.

Kinh nghiệm làm việc

Dự án cá nhân

  • Phát triển ứng dụng web sử dụng Ruby on Rails với Stripe API để xử lý thanh toán.
  • Triển khai OmniAuth để xác thực người dùng thông qua các mạng xã hội.
  • Hỗ trợ nhóm trong việc kiểm tra và sửa lỗi ứng dụng.
  • Mô hình nhà thông minh: Lập trình C++ cho hệ thống nhúng điều khiển các thiết bị trong nhà, giúp tối ưu hóa việc sử dụng năng lượng.
  • Máy sấy đồ tự động: Phát triển ứng dụng nhúng điều khiển máy sấy đồ, tích hợp cảm biến để tự động điều chỉnh nhiệt độ.
  • Game với SFML: Phát triển game sử dụng SFML, tạo đồ họa và cơ chế chơi game hấp dẫn.
  • Ứng dụng web quản lý thời gian: Phát triển ứng dụng web cho phép người dùng quản lý và theo dõi thời gian làm việc, sử dụng Ruby on Rails.
